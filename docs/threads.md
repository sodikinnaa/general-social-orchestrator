# Threads Posting API Documentation

Dokumentasi ini merinci penggunaan endpoint Threads pada **General Social Publisher API**.

## Endpoint: Post to Threads
Digunakan untuk mengirim status teks atau foto ke akun Threads.

- **Method:** `POST`
- **URL:** `{{baseUrl}}/api/v1/threads/post`
- **Headers:**
  - `Content-Type: application/json`
  - `x-license-key`: (Optional) Sultan License Key.
  - `x-device-id`: (Optional) Hardware ID.

### 1. Payload: Text Only
Kirim status berupa teks saja (Maksimal 500 karakter).
```json
{
    "access_token": "YOUR_THREADS_ACCESS_TOKEN",
    "text": "Hello Threads! Ini adalah postingan teks pertama dari API Sultan Engine."
}
```

### 2. Payload: Image with Text
Kirim satu foto dengan teks/caption.
```json
{
    "access_token": "YOUR_THREADS_ACCESS_TOKEN",
    "text": "Caption foto keren di Threads!",
    "image_url": "https://example.com/photo.jpg"
}
```

## Endpoint: Verify Token
Digunakan untuk memvalidasi apakah Threads Access Token Anda aktif dan melihat informasi profil yang terhubung.

- **Method:** `POST`
- **URL:** `{{baseUrl}}/api/v1/threads/verify`
- **Payload:**
```json
{
    "access_token": "YOUR_THREADS_ACCESS_TOKEN"
}
```

## 🛠️ Cara Mendapatkan Threads Access Token

Threads menggunakan sistem token yang berbeda dari Facebook Graph API. Ikuti langkah ini:

1.  Buka [Meta for Developers](https://developers.facebook.com/) dan pilih aplikasi Anda.
2.  Tambahkan Use Case **"Akses Threads API"** melalui menu **"Use Cases"**.
3.  Di sidebar kiri, masuk ke menu **Threads** -> **API Setup**.
4.  Tambahkan akun Threads Anda di tabel **"Threads Account for Testing"**.
5.  Otorisasi aplikasi melalui aplikasi Threads di HP Anda (**Settings -> Account -> Website Permissions -> Approve**).
6.  Klik tombol **"Generate Token"** di portal Meta Developer.
7.  **Penting:** Token Threads yang valid biasanya memiliki format yang berbeda dari token Facebook (User/Page).

---

## 🛠️ Izin & Fitur Utama (Permissions)

Buka Meta Developer Portal -> Use Cases -> Customize -> **Akses Threads API** -> Izin dan fitur. Tambahkan izin berikut:

1.  **`threads_content_publish`** (Wajib): Untuk membuat postingan baru.
2.  **`threads_basic`** (Wajib): Untuk membaca informasi dasar profil.
3.  **`threads_manage_replies`**: (Opsional) Untuk membalas atau mengelola komentar.

---

## ⚠️ Batasan & Tips
- **Limit Karakter:** Threads mendukung hingga 500 karakter per post.
- **Media:** Saat ini endpoint ini mendukung satu gambar per post via `image_url`. Pastikan URL gambar dapat diakses secara publik.
- **Error 401 (Unauthorized):** Biasanya terjadi jika token salah atau bukan berasal dari Threads API (misal: malah memasukkan token Facebook).

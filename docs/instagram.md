# Instagram Posting API Documentation

Dokumentasi ini merinci penggunaan endpoint Instagram pada **General Social Publisher API**.

## Endpoint: Post to Feed
Digunakan untuk mengirim status berupa foto ke feed Instagram Business.

- **Method:** `POST`
- **URL:** `{{baseUrl}}/api/v1/instagram/post`
- **Headers:**
  - `Content-Type: application/json`

### Payload: Single Photo
```json
{
    "ig_user_id": "YOUR_INSTAGRAM_BUSINESS_ID",
    "access_token": "YOUR_PAGE_ACCESS_TOKEN",
    "caption": "Hello Instagram! Posting dari API Sultan Engine.",
    "image_url": "https://example.com/photo.jpg"
}
```

## 🛠️ Cara Mendapatkan Instagram Business ID

Instagram Business API terhubung melalui Facebook Page. Ikuti langkah ini:

1.  **Pastikan Akun IG adalah Bisnis/Kreator** dan sudah dihubungkan ke sebuah Facebook Page.
2.  Gunakan endpoint Facebook kita: `POST {{baseUrl}}/api/v1/facebook/pages`.
3.  Masukkan **User Access Token** Anda.
4.  Cari Fanspage yang terhubung dengan Instagram Anda.
5.  Lihat field **`instagram_business_account`**. Di sana akan ada **`id`** (Inilah `ig_user_id` Anda).
6.  Gunakan `id` tersebut dan `access_token` milik Fanspage tersebut untuk posting ke Instagram.

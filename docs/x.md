# X (Twitter) Posting API Documentation

Dokumentasi ini merinci penggunaan endpoint X pada **General Social Publisher API**.

## Endpoint: Post Tweet
Digunakan untuk mengirim Tweet berupa teks atau Tweet dengan media (gambar).

- **Method:** `POST`
- **URL:** `{{baseUrl}}/api/v1/x/post`
- **Headers:**
  - `Content-Type: application/json`

### 1. Payload: Text Only
```json
{
    "api_key": "YOUR_X_API_KEY",
    "api_secret": "YOUR_X_API_SECRET",
    "access_token": "YOUR_X_ACCESS_TOKEN",
    "access_secret": "YOUR_X_ACCESS_SECRET",
    "text": "Hello X! Posting otomatis via Sultan Engine Core."
}
```

### 2. Payload: Tweet with Media
API mendukung pengiriman hingga 4 gambar (limit dari X). Mas cukup memasukkan array URL gambar.
```json
{
    "api_key": "...",
    "api_secret": "...",
    "access_token": "...",
    "access_secret": "...",
    "text": "Tweet dengan gambar keren!",
    "media_urls": [
        "https://example.com/image1.png"
    ]
}
```

## 🛠️ Cara Mendapatkan & Pemetaan Kredensial X

Untuk menggunakan API ini, Anda memerlukan 4 kunci utama dari [X Developer Portal](https://developer.twitter.com/en/portal/dashboard). Berikut adalah tabel pemetaannya:

| Field di JSON Payload | Nama di X Developer Console | Lokasi di Tab "Keys and Tokens" |
| :--- | :--- | :--- |
| **`api_key`** | **API Key** | Kotak **"Consumer Keys"** |
| **`api_secret`** | **API Key Secret** | Kotak **"Consumer Keys"** |
| **`access_token`** | **Access Token** | Kotak **"Authentication Tokens"** |
| **`access_secret`** | **Access Token Secret** | Kotak **"Authentication Tokens"** |

### Langkah Pengambilan Key:

1.  **Ubah Izin Aplikasi (Wajib):** 
    *   Masuk ke **User authentication settings** -> **Set up**.
    *   Ubah App Permissions ke **Read and Write**.
    *   Simpan pengaturan.
2.  **Generate Consumer Keys:** 
    *   Klik **Regenerate** pada bagian Consumer Keys untuk mendapatkan `api_key` dan `api_secret`.
3.  **Generate Access Tokens:** 
    *   Klik **Generate** (atau Regenerate) pada bagian Authentication Tokens untuk mendapatkan `access_token` dan `access_secret`.
    *   *Catatan: Jika Anda baru saja mengubah izin ke Read/Write, Anda harus melakukan Regenerate Access Token agar izin barunya aktif.*

## ⚠️ Batasan & Tips
- **Rate Limit:** Akun Free X API memiliki limit yang sangat ketat (sekitar 15 post per 24 jam untuk level Free v2).
- **Media Upload:** API Sultan secara otomatis mengunduh gambar dari URL yang Mas berikan dan mengunggahnya ke server X sebelum posting.
- **Error 403 (Forbidden):** Biasanya terjadi jika App Permissions belum diubah ke "Read and Write".

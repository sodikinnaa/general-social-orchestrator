# INSTALLATION GUIDE: OpenClaw Plugin Bridge
## General Social Orchestrator (v1.0.0)

Plugin ini adalah bridge yang menghubungkan instruksi AI dengan **General Social Publisher API**.

---

## 🏗️ Quick Setup

### 1. Installation
Clone repository ke dalam folder plugin OpenClaw Anda:
```bash
cd ~/.openclaw/workspace/plugins
git clone git@github.com:sodikinnaa/general-social-orchestrator.git
cd general-social-orchestrator
npm install
```

### 2. Manual Configuration (Auth)
Untuk saat ini, buat file `config.json` di root folder plugin ini untuk menyimpan kredensial platform:
```json
{
  "fb": {
    "page_id": "YOUR_PAGE_ID",
    "access_token": "YOUR_ACCESS_TOKEN"
  },
  "x": {
    "api_key": "...",
    "api_secret": "...",
    "access_token": "...",
    "access_secret": "..."
  },
  "threads": {
    "access_token": "..."
  }
}
```

### 3. Usage via OpenClaw
Setelah terinstal, AI akan memiliki akses ke tool:
`social_orchestrator_publish`

**Contoh Command AI:**
> "Post ke Facebook dan Twitter: Halo Dunia!"

## 🔗 Technical Bridge Details
- **Target API:** `http://brain.siapdigital.my.id:19092`
- **Orchestrator Endpoint:** `/api/v1/orchestrator/publish`
- **Method:** `POST`

## 🛠️ Troubleshooting
- **Missing config.json:** Plugin akan gagal eksekusi jika file config tidak ditemukan.
- **API Offline:** Pastikan Backend API di server Sultan (Port 19092) dalam kondisi aktif.
- **JSON Error:** Pastikan `media_urls` dikirim dalam format array string.

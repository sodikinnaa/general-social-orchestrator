# INSTALLATION GUIDE: OpenClaw Plugin Bridge
## General Social Orchestrator (v1.1.0)

Plugin ini adalah bridge yang menghubungkan instruksi AI dengan **General Social Publisher API**.

---

## 🏗️ Quick Setup

### 1. Installation
Instal via NPM atau clone repository:
```bash
# Via NPM
npm install @sodikinnaa/general-social-orchestrator

# Atau via Git
cd ~/.openclaw/workspace/plugins
git clone https://github.com/sodikinnaa/general-social-orchestrator.git
```

### 2. Native Configuration (Recommended)
Kredensial disimpan secara aman di `openclaw.json`. Masukkan kredensial melalui dashboard OpenClaw atau edit file config Anda:

```json
"general-social-orchestrator": {
  "enabled": true,
  "config": {
    "fb": { "page_id": "...", "access_token": "..." },
    "x": { "api_key": "...", "api_secret": "...", "access_token": "...", "access_secret": "..." },
    "threads": { "access_token": "..." }
  }
}
```

### 3. Usage via OpenClaw
Setelah terinstal, AI akan memiliki akses ke tool:
`social_orchestrator_publish`
